Hello World Project

Description: The Hello World Project was the first project assigned to show the students the basic shell of an HTML program.

Resume

Description: This Resume Site was assigned to allow the students to showcase their work experience while also allowing them to experiment with HTML and CSS for the first time.

Bio Page

Description: The Bio Page was used to teach the students how to properly use CSS Selectors to provide the basic structure of front end web design.

Blog Layout

Description: With the Blog Site, the students created static examples of a blog form again using CSS Selectors to format multiple objects with one call in the stylesheet.

UX Airplane Site

Description: The UX Airplane Site build was the first time the students were asked to emulate an already built site using only previously given images and a model of what the finished site should look like. The purpose was to master the Box Model concept in HTML.

NASA

Description: The NASA webpage was again another emulation assignment however this time the challenge was to use the Flexbox layout and make sure the site matched the given dimensions.

Welcome Site

Description: The Welcome Site was used to demonstrate the use of media queries to build responsive web pages whether it was being viewed from a laptop, phone, or tablet.

News Website

Description: The News Website introduced to the students the use of the language Bootstrap. With this implementation the students learned a much easier method to go about creating responsive web pages.

Transitions and Transformations

Description: Transitions and Transformations was a fun project assigned to allow the students to play around with the CSS code and and show how these transition and transform features can be used to enhance a web page. The students also learned they can be overkill if used too much.

Dice Game

Description: The Dice Game program was the introduction for JavaScript, If/Else statements, and For loops. The students used all previous knowledge to test and output the die val in a version of Yachtzee.

Calculator

Description: Upon digging deeper into JavaScript, the students learned the use of functions to create a basic calculator that return a value of the user's arithmetic.

Dom Practice

Description: The Shopping Cart page had the students use the Document Object Model to target and insert elements in HTML solely from a JavaScript file.
